# klm-release

#### 介绍

Kugou Lyrics Maker 发布页

krc、lrc歌词制作工具，本工具完全免费，可以制作酷狗krc歌词，也能保存成lrc歌词。更多方便的功能请在软件中体验

#### 版本更新

##### 1.0.0.21

1.  new: 读音/翻译双语歌词制作功能

##### 1.0.0.20

1.  fix: 后缀名判断优化，避免大写后缀、大小写混合后缀无法正常处理问题
2.  fix: 修正为保存lrc时，首行0开始才加入\[00:00.00\]的空行
