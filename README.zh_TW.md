# klm-release

#### 介紹

Kugou Lyrics Maker 發布頁

krc、ksc、lrc歌詞製作工具，本工具完全免費，可以製作酷狗krc歌詞，也能保存成lrc歌詞。更多方便的功能請在軟件中體驗

#### 版本更新

##### 1.0.3.2

2.  fix: 轉換功能，krc保存位置無法被清空問題

1.  new: 格式互轉功能。支持\*.krc \*.ksc \*.lrc(標準lrc/精準ESLyric 0.3.x lrc/精準ESLyric 0.5.x lrc)作為源格式進行轉換
